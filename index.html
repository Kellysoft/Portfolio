Dojo Overflow
Dojo Overflow was built to recreate much of the functionally of Reddit. DO uses Socket.io for real-time communication, integration of Facebook login API, and front back and middleware validations. With minimal styles, this is a feature driven app. Users get the full discussion board effect with comments, replies, and threads. Both comments and threads can be upvoted and downvoted which effect the users "Karma Score."
